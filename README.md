# 🐚 Abalone Age Predictor

A full-stack ML web app that predicts abalone age from physical measurements
using a Decision Tree Regressor trained on the UCI Abalone dataset.

---

## 📁 Project Structure

```
abalone-app/
├── backend/
│   ├── model/                 ← pickle file goes here (generated by script)
│   ├── app.py                 ← Flask API
│   ├── train_and_save.py      ← trains model and saves pickle
│   ├── requirements.txt       ← Python dependencies
│   ├── Procfile               ← for Render deployment
│   └── abalone.csv            ← ⚠ YOU must copy this here (see Step 1)
└── frontend/
    ├── src/
    │   ├── App.jsx            ← React UI
    │   └── main.jsx           ← React entry point
    ├── index.html
    ├── package.json
    └── vite.config.js
```

## 🚀 How to Run Locally

### Prerequisites
- Python 3.9+ installed
- Node.js 18+ installed
- Your `abalone.csv` dataset file

---

### Step 1 — Add the CSV file

Copy your `abalone.csv` file into the `backend/` folder:

```
abalone-app/
└── backend/
    └── abalone.csv   ← paste it here
```

---

### Step 2 — Set up Python backend

Open a terminal and run:

```bash
cd abalone-app/backend

# Create a virtual environment (recommended)
python -m venv venv

# Activate it
# On Windows:
venv\Scripts\activate
# On Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

---

### Step 3 — Train the model and save pickle

```bash
# Make sure you're inside backend/ with venv activated
python train_and_save.py
```

You should see:
```
✅  Dataset loaded: 4177 rows, 9 columns
✅  Train size: 3341 | Test size: 836
✅  Decision Tree → MSE: 9.xxxx | R²: 0.xxxx
✅  Model saved to model/abalone_model.pkl
🎉  Done! Now run: python app.py
```

---

### Step 4 — Start the Flask backend

```bash
python app.py
```

Backend is now running at: **http://localhost:5000**

Test it in your browser: http://localhost:5000
→ Should show: `{"status": "Abalone API is running 🐚"}`

---

### Step 5 — Set up and start the React frontend

Open a **new terminal** (keep the backend running):

```bash
cd abalone-app/frontend

# Install Node dependencies
npm install

# Start development server
npm run dev
```

Frontend is now running at: **http://localhost:3000**

Open http://localhost:3000 in your browser — the app is live!

---

## 🧪 How to Use the App

1. Select the **sex** of the abalone (Male / Female / Infant)
2. Enter all 7 measurements (see the range hints below each field)
3. Click **Predict Ring Count**
4. The app shows:
   - Number of rings
   - Estimated age (rings + 1.5 years)

---

## 🌐 Deploying to Production

### Backend → Render.com

1. Push the `backend/` folder to GitHub (include `model/abalone_model.pkl`)
2. Go to render.com → New Web Service
3. Connect your GitHub repo
4. Set:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
5. Deploy → copy your live URL

### Frontend → Update API URL

In `frontend/src/App.jsx`, change line 7:

```js
// Before (local dev)
const API_URL = "/predict";

// After (production)
const API_URL = "https://your-app-name.onrender.com/predict";
```

Then deploy frontend to Vercel or Netlify.

## 📊 Model Details

| Property | Value |
|---|---|
| Algorithm | Decision Tree Regressor |
| Dataset | UCI Abalone (4177 samples) |
| Features | Sex, Length, Diameter, Height, Whole Weight, Shucked Weight, Viscera Weight, Shell Weight |
| Target | Rings (Age = Rings + 1.5 years) |
| Train/Test Split | 80% / 20% |
| Sex Encoding | M=0, F=1, I=2 |
